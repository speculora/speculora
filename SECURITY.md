# Security Policy

This repository contains **documentation only** — no source code, secrets, or
production infrastructure. There is no runnable software here to exploit.

## Reporting a vulnerability

If you discover a security issue with the Speculora product or website
(https://speculora.com) — not this docs repository — please report it
responsibly:

- Email **hello@speculora.com** with a clear description and steps to reproduce.
- Please do not open a public issue for security-sensitive reports.

We aim to acknowledge reports within a few business days. We appreciate
responsible disclosure and will credit reporters who wish to be named once an
issue is resolved.

## Documentation issues

For factual errors, broken links, or unclear content in this repository, open a
normal issue or pull request — see [CONTRIBUTING.md](CONTRIBUTING.md).
